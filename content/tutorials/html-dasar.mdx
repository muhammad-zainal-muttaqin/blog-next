HTML adalah kerangka dasar halaman web. Fokus ke elemen inti terlebih dulu.

1) Struktur minimum

```html
<!doctype html>
<html lang="id">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>Halo</title>
  </head>
  <body>
    <h1>Halo Dunia</h1>
    <p>Ini paragraf pertama.</p>
  </body>
  </html>
```

2) Teks dan tautan

```html
<h1>Judul</h1>
<p>Tulisan biasa dengan <a href="https://example.com">tautan</a>.</p>
```

3) Gambar dan daftar

```html
<img src="/logo.png" alt="Logo" />
<ul>
  <li>Satu</li>
  <li>Dua</li>
  <li>Tiga</li>
</ul>
```

Kiat KISS: mulai dari elemen fundamental. Tambahkan yang lain saat diperlukan.
