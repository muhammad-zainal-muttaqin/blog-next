## Mengapa MDX?

MDX memungkinkan kita menulis konten Markdown yang bisa menyisipkan komponen React. Hasilnya: konten tetap ringan, tapi bisa interaktif dan konsisten secara desain.

## Anchor Heading Otomatis

Setiap `h2`/`h3` sekarang memiliki anchor otomatis. Coba klik simbol ¶ di sebelah judul untuk menyalin tautan bagian ini.

## Callout Komponen

<Callout>
Tips: Gunakan Callout untuk menekankan hal penting tanpa memecah fokus pembaca.
</Callout>

## Contoh Kode

```ts
// util/slug.ts
export function toSlug(title: string): string {
  return title
    .toLowerCase()
    .replace(/[^a-z0-9\s-]/g, "")
    .trim()
    .replace(/\s+/g, "-");
}
```

## Penutup

Dengan fondasi ini, konten lebih nyaman dibaca, konsisten, dan mudah dinavigasi. Selanjutnya kita bisa menambah komponen interaktif (demo, sandpack, dsb.) untuk pengalaman belajar yang lebih kaya.
