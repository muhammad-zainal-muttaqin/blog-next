Tujuan: membangun landing page modern yang rapi, performa baik, aksesibel, dan siap produksi. Kita gabungkan seluruh dasar HTML, CSS, dan JavaScript yang sudah dipelajari, lalu sedikit "overengineering" untuk rasa profesional.

Hasil akhir mencakup:
- Header sticky + active link (scrollspy)
- Hero dengan gradient + glassmorphism
- Grid cards (fitur) + hover 3D subtle
- Testimonial carousel sederhana
- FAQ accordion
- Contact form dengan validasi
- Dark mode toggle (persist ke `localStorage`)
- Animasi masuk konten saat scroll (IntersectionObserver)

Struktur direktori minimal:

```text
public/
  images/
    avatar-1.jpg
    avatar-2.jpg
index.html
styles.css
app.js
```

3) JavaScript: theme, scrollspy, reveal, carousel, validasi form

```js
// theme toggle
const themeToggle = document.getElementById('themeToggle');
const root = document.documentElement;
const savedTheme = localStorage.getItem('theme');
if (savedTheme) root.classList.toggle('light', savedTheme === 'light');
themeToggle?.addEventListener('click', () => {
  const isLight = root.classList.toggle('light');
  localStorage.setItem('theme', isLight ? 'light' : 'dark');
});

// scrollspy aktifkan link berdasarkan section terlihat
const links = Array.from(document.querySelectorAll('.nav-link'));
const sections = links.map(l => document.querySelector(l.getAttribute('href')));
const spy = new IntersectionObserver((entries) => {
  for (const entry of entries) {
    const idx = sections.indexOf(entry.target);
    if (idx >= 0) {
      links[idx].classList.toggle('active', entry.isIntersecting);
    }
  }
}, { rootMargin: '-40% 0px -55% 0px', threshold: 0.01 });
sections.forEach(s => s && spy.observe(s));

// reveal elements
const reveals = document.querySelectorAll('.reveal');
const io = new IntersectionObserver((entries) => {
  entries.forEach((e) => e.target.classList.toggle('revealed', e.isIntersecting));
}, { threshold: 0.12 });
reveals.forEach(el => io.observe(el));

// tilt card subtle
document.querySelectorAll('.tilt').forEach(card => {
  card.addEventListener('mousemove', (e) => {
    const r = card.getBoundingClientRect();
    const rx = (e.clientY - r.top) / r.height - 0.5;
    const ry = (e.clientX - r.left) / r.width - 0.5;
    card.style.transform = `rotateX(${rx * -4}deg) rotateY(${ry * 4}deg)`;
  });
  card.addEventListener('mouseleave', () => { card.style.transform = ''; });
});

// carousel
const track = document.getElementById('carouselTrack');
document.getElementById('prev')?.addEventListener('click', () => track?.scrollBy({ left: -300, behavior: 'smooth' }));
document.getElementById('next')?.addEventListener('click', () => track?.scrollBy({ left: 300, behavior: 'smooth' }));

// form validation
const form = document.getElementById('contactForm');
const note = document.getElementById('formNote');
form?.addEventListener('submit', (e) => {
  e.preventDefault();
  const fd = new FormData(form);
  const name = String(fd.get('name') || '').trim();
  const email = String(fd.get('email') || '').trim();
  const message = String(fd.get('message') || '').trim();
  if (name.length < 3) return (note.textContent = 'Nama minimal 3 karakter');
  if (!/^[^@\s]+@[^@\s]+\.[^@\s]+$/.test(email)) return (note.textContent = 'Format email tidak valid');
  if (message.length < 10) return (note.textContent = 'Pesan terlalu singkat');
  note.textContent = 'Terkirim! (simulasi)';
});
```
